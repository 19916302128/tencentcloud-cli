**Example 1: Disabling workflow**

This example shows you how to disable the workflow whose ID is 19000.

Input: 

```
tccli mps DisableWorkflow --cli-unfold-argument  \
    --WorkflowId 19000
```

Output: 
```
{
    "Response": {
        "RequestId": "6ca31e3a-6b8e-4b4e-9256-fdc700064ef3"
    }
}
```
