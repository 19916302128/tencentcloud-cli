**Example 1: Enabling workflow**

This example shows you how to enable the workflow whose ID is 10000.

Input: 

```
tccli mps EnableWorkflow --cli-unfold-argument  \
    --WorkflowId 10000
```

Output: 
```
{
    "Response": {
        "RequestId": "6ca31e3a-6b8e-4b4e-9256-fdc700064ef3"
    }
}
```
