**Example 1: Modifying the name of a snapshot**



Input: 

```
tccli cbs ModifySnapshotAttribute --cli-unfold-argument  \
    --SnapshotId snap-gybrif0z \
    --SnapshotName snap_201711301143
```

Output: 
```
{
    "Response": {
        "RequestId": "55db49cf-b9d7-da27-825b-5a02ba6884ca"
    }
}
```
