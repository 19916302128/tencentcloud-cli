**Example 1: Creating queue**

This example shows you how to create a queue.

Input: 

```
tccli cmq CreateQueue --cli-unfold-argument  \
    --QueueName test
```

Output: 
```
{
    "Response": {
        "QueueId": "queue-ges05csc",
        "RequestId": "3e0dff9d-9ed5-47c3-beb2-a42c1d69e1cc"
    }
}
```
