**Example 1: Deleting a policy version**

This example shows you how to delete a policy version of a policy.

Input: 

```
tccli cam DeletePolicyVersion --cli-unfold-argument  \
    --PolicyId 17698703\
    --VersionId 1
```

Output: 
```
{
    "Response": {
        "RequestId": "60e60a86-af67-4bbe-8377-7024a0e1d4c7"
    }
}
```
