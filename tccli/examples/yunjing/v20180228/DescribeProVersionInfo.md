**Example 1: Getting CWP Pro information**

This example shows you how to get the CWP Pro information.

```
tccli yunjing DescribeProVersionInfo --cli-unfold-argument ```

        "PostPayCost": 4,
        "IsAutoOpenProVersion": false,

```
