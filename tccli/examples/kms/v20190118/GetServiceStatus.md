**Example 1: Querying service status**

This example shows you how to query the service status.

Input: 

```
tccli kms GetServiceStatus --cli-unfold-argument ```

Output: 
```
{
    "Response": {
        "ServiceEnabled": true,
        "InvalidType": 1,
        "RequestId": "1b580852-1e38-11e9-b129-5cb9019b4b00"
    }
}
```
