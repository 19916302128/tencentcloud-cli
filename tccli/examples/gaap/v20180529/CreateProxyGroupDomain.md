**Example 1: Enabling a Connection Group Domain Name**

```
tccli gaap CreateProxyGroupDomain --cli-unfold-argument  \
    --GroupId lg-b7h4d02f
```

{

        "RequestId": "19a021f8-dff3-4890-8e7a-ed5054e22e49",
        "GroupId": "lg-okl0h0n1"

}
