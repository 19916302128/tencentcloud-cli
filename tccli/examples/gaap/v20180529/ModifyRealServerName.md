**Example 1: Modifying an Origin Server Name**

Modify an origin server name.

Input: 

```
tccli gaap ModifyRealServerName --cli-unfold-argument  \
    --RealServerName ‘namexxx’\
    --RealServerId 'rsxxxx'
```

Output: 
```
{
    "Response": {
        "RequestId": "5c680029-66b2-4be8-9630-7bd316ce70dd"
    }
}
```
