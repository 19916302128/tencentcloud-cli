**Example 1: Modifying the Connection Attributes**



Input: 

```
tccli gaap ModifyProxiesAttribute --cli-unfold-argument  \
    --ProxyIds link-11112222 link-11113333 \
    --ProxyName test
```

Output: 
```
{
    "Response": {
        "RequestId": "04d1af8f-5c76-4053-acbc-62bdf0541c15"
    }
}
```
